### Python Set Up:

#### Create a Virtual Environment
```
python -m venv venv
```

#### Activate VENV
```
.\venv\Scripts\activate
```

#### Install libraries in VENV
```
cd ml
pip install -r requirements.txt
```

#### In Terminal 1: Run Python Main File
```
python app.py
```

#### Create React App
Tailwind
React Icons
Rechart
Lucid Icons 
React Router Dom were used
npm install recharts lucide-react date-fns
```
npm i
```

#### In Terminal 2: Run React App 
```
cd frontend
npm run dev
```

#### In Terminal 3: Test Bots
```
python bot_smart.py
```
